Detailed Description

A custom floating-point class with configurable precision and a "hub" bit for consistent rounding.

The hub_float class implements a floating-point format that uses a special "hub" bit. Internally, values are stored as doubles that are quantized to lie on a specific grid determined by the exponent and mantissa and the extra "hub" bit, an implicit least significant bit.
